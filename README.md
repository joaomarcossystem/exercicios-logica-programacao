# Teste Técnico - Lógica de Programação

Este repositório contém a minha solução para a avaliação de conhecimento técnico. 
Optei por criar uma única Solução no Visual Studio, utilizando linguagem C#, separando os três exercícios em projetos de Console independentes para deixar tudo mais organizado.

## Estrutura dos Exercícios

* **Exercício 1 (Calculadora Salarial):** Recebe os dados de um empregado e calcula o salário bruto, o salário família (baseado nas faixas de renda e filhos menores que 14 anos) e o salário líquido.
* **Exercício 2 (Sequência Numérica):** O usuário escolhe o tamanho de uma sequência, digita os números, e o sistema guarda em um vetor (array) para descobrir qual foi o maior e o menor número digitado.
* **Exercício 3 (Lógica de Fibonacci):** Imprime a sequência de Fibonacci de acordo com a quantidade pedida e avisa se o número escolhido faz parte ou não da sequência matemática.

## Tecnologias Utilizadas
* C#
* .NET 10.0
* Visual Studio 2022

## Como rodar o código
1. Faça o download do projeto.
2. Abra o arquivo `AvaliacaoTecnicaAcedata.slnx` no seu Visual Studio.
3. No Gerenciador de Soluções, clique com o botão direito no projeto do exercício que deseja testar e escolha **"Definir como Projeto de Inicialização"**.
4. Clique em Iniciar.
